== Custom Cam

Custom Cam is a weekend project developed by 4 students during their 7th weekend at Dev Bootcamp. 

It is a crowdsourced marketplace for photo contests that connects freelance photographers to instant jobs.

Created by: David Wen (Github)[https://www.github.com/dawenster], Gavin St. Ours (Github)[https://www.github.com/charmcitygavin], Kevin Ko (Github)[https://www.github.com/heyimkko], Philip Woo (Github)[https://www.github.com/jhw1202]
