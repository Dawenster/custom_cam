== Custom Cam

Custom Cam is a weekend project developed by 4 students during their 7th weekend at Dev Bootcamp. 

It is a crowdsourced marketplace for photo contests that connects freelance photographers to instant jobs.

Created by: <a href="https://www.github.com/dawenster">David Wen</a>, [Gavin St. Ours](https://www.github.com/charmcitygavin), [Kevin Ko](https://www.github.com/heyimkko), [Philip Woo](https://www.github.com/jhw1202)
